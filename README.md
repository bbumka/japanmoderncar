# Japan Modern Car

このプロジェクトは、日本で新しく製造された自動車を検索・販売するためのウェブサイトです。

## 主な機能
- 日本の最新モデルの自動車検索
- 車両の詳細情報表示
  

## 使用技術
-React 18	UI component
Vite	Build tool, dev server (localhost:5173)
CSS 
JavaScript (ES6+)	
Google Gemini AI	 (generate_images.py)
Node.js / npm	Package manage
